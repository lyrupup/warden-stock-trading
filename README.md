# 守望者股票交易系统（Warden Stock Trading System）

> 🦉 一套面向个人投资者的线上股票辅助决策与交易管理 Web 系统。
>
> 盘前有预案，盘中有监控，盘后有复盘 —— 让 AI 成为你的私人投研守望者。

---

## ✨ 灵感来源

项目命名与设计灵感来源于《魔兽争霸 III》中以「侦查、洞察、警戒、追猎」著称的暗夜精灵英雄 —— **守望者（Warden）**。

我们将其特质映射为系统的四大「技能彩蛋」，贯穿各功能模块：

| 技能 | 代号 | 对应能力 |
|------|------|---------|
| 🔍 侦查 | **Scan** | 拉取实时行情、持仓数据 |
| 🧠 洞察 | **Insight** | AI 深度分析、趋势判断 |
| 🚨 警戒 | **Alert** | 异动风险提醒、大跌告警 |
| 📨 传信 | **Message** | 飞书报告推送 |

---

## 🧩 功能模块

| 模块 | 能力 | 简介 |
|------|------|------|
| M1 大盘与 A 股行情数据 | Scan | 每日自动展示大盘 / A 股行情，自选个股行情拉取 |
| M2 个人选股策略 | Insight | 自定义策略（描述 + 指标 + skill.md + 回测） |
| M3 个人持仓记录 | Scan | 持仓管理、买卖记录、自动盈亏计算 |
| M4 持仓分析与风控 | Insight + Alert | AI 每日持仓分析、风险提示、明日盘前预案 |
| M5 个人定时任务 | Alert + Message | 定时脚本 / 报告，推送飞书云文档与机器人 |
| M6 AI 对话分析 | Insight | 用选股策略对个股做 AI 分析，报告按股票代码归档 |
| M7 个人中心配置 | 基础 | 大模型 / 飞书 API Key、白天暗夜模式等个人配置 |

> 系统当前以单用户方式使用，但功能与数据模型已为**多用户扩展**预留（所有数据按 `user_id` 隔离）。

---

## 📐 技术方案（建议）

> 详见 PRD 第 7 章，最终以技术开发文档为准。

- **前端**：React + Vite + shadcn/ui + Tailwind CSS，ky + TanStack Query，zustand，支持 Light/Dark 主题
- **后端**：Go + Gin + GORM，含限流 / 超时 / 鉴权 / 日志中间件
- **存储**：PostgreSQL + Redis（可选 RabbitMQ）
- **集成**：行情数据源、飞书开放平台、大模型 API
- **部署**：Docker + docker-compose，配置经环境变量注入
